AI Honeypot (Cowrie + Docker)

A fake SSH honeypot powered by Cowrie, containerized with Docker to safely capture attacker behavior for research and education.

⭐ View on GitHub Port 2222 • JSON logs • Local-first

Features

  • ✅ Fake SSH server on 2222
  • ✅ Captures logins, commands, and file downloads (wget/curl)
  • ✅ JSON log output for easy analysis
  • ✅ Dockerized — quick setup & teardown

Quick Start

# 1) Clone & enter
git clone https://github.com/saran-1305/ai-honeypot.git
cd ai-honeypot

# 2) Start Cowrie
docker compose -f docker-compose.cowrie.yml up -d

# 3) Connect (enter any password)
ssh -o StrictHostKeyChecking=no -p 2222 root@127.0.0.1

# 4) Follow logs
tail -f cowrie-logs/cowrie.json

Example JSON event

{
  "eventid": "cowrie.session.connect",
  "src_ip": "192.168.1.10",
  "username": "root",
  "password": "admin",
  "command": "uname -a",
  "timestamp": "2025-09-20T12:00:00Z"
}

Notes

For learning & demos. Do not expose directly to the public internet unless you know what you’re doing. Honeypots attract real attackers—use a lab / cloud sandbox.
